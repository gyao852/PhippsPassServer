Pass Data

Overview page of all passes held in the server. A 'Pass' holds the meta data of the digital membership cards. The 'Serial Number' is a unique number value associated with each membership card. Moreover, the 'Authentication Token' is a value that is repeatedly checked against the local database for valdiations; it is also unique for each pass created. The 'File Name' is simply a concatenation of the first and last name for that member. 'Last Sent' refers to when an email was last sent to a member, or when it was last updated (this is because, after the pass is installed on a device, each time Apple Wallet or Google Play App 'Pass' is opened, it will check against our server for the latest update automatically. Note that for every record in membership, there can be zero to multiple passes associated with it. If a record in the Membership table is currently NOT a member, then he/she will NOT have a pass created.
